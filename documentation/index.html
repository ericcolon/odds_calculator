Installing

You can install it via pip:

pip install oddscalculator

Usage

To use oddscalculator, simply do: 

from oddscalculator import *

odds = DecimalOdds(1.72)
print odds.decimal
print odds.american
print odds.malay
print odds.indonesian
print odds.hongkong

odds = AmericanOdds(-139)
print odds.decimal
print odds.american
print odds.malay
print odds.indonesian
print odds.hongkong
